repeating-linear-gradient()

  • 版本：CSS3

语法：

<repeating-linear-gradient>：repeating-linear-gradient( [ <angle> | to <side-or-corner> ,]? <color-stop> [, <color-stop>]+ )

<repeating-linear-gradient>(私有前缀旧语法)：-prefix-repeating-linear-gradient( [ <angle> | <side-or-corner> ,]? <color-stop> [, <color-stop>]+ )

<side-or-corner>：[left | right] || [top | bottom]

<color-stop>：<color> [ <percentage> | <length> ]?

取值：

<side-or-corner>：
表示开始点的梯度线的位置。它由两个关键字：第一个指示水平方向，left或right，和第二个垂直方向，top或bottom。与顺序无关，每一个关键字都是可选的。to top, to bottom, to left和to right的取值分别转换为角0度，180度，270deg，90度。其他取值会被翻译成可让起始点处于该取值所描述的象限，同时垂直于相邻象限对角线的角度。这样一来，由<color-stop>描述的颜色正好适用于角点。这有时被称为“魔角”属性。结束点会相对正中心，与起始点对称。
<angle>：
用角度值指定渐变的方向（或角度）。0deg为由下往上，90deg为从左往右，以此类推。注意私有前缀旧语法中，0deg为从左往右，90deg为从下往上。

<color-stop>

<color>：
指定颜色。
<length>：
用长度值指定起止色位置。不允许负值
<percentage>：
用百分比指定起止色位置。

说明：

用重复的线性渐变创建图像。
  • repeating-linear-gradient()的语法与linear-gradient()相同。

  • 示例代码：

    （图一）

    repeating-linear-gradient(#f00,#ff0 10%,#f00 15%);
repeating-linear-gradient(180deg,#f00,#ff0 10%,#f00 15%);

    以上几句代码都可以实现如（图一）的渐变效果

兼容性：

  • 浅绿 = 支持
  • 红色 = 不支持
  • 粉色 = 部分支持
